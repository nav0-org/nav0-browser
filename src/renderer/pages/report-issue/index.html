Technical Details

Log Files

Log files are extremely helpful for diagnosing issues. Here's how to find and attach them:

Finding Log Files

  • Windows:
    C:\Users\[YourUsername]\AppData\Local\Nav0 Browser\Logs
  • macOS:
    ~/Library/Application Support/Nav0 Browser/Logs
  • Linux:
    ~/.config/nav0-browser/logs

Generating Diagnostic Reports

  1. Open Nav0 Browser
  2. Click on the menu icon (three dots in the top-right corner)
  3. Select "Help" > "Diagnostic Report"
  4. Click "Generate Report"
  5. Save the report file and attach it to your email

Note: Log files and diagnostic reports do NOT contain your browsing history, passwords, or personal data. They only include technical information about the browser's operation and any errors that occurred.

Tips for Effective Bug Reports

Be Specific and Detailed

The more specific you are about the issue, the easier it will be for our team to reproduce and fix it. Vague descriptions like "the browser crashed" are less helpful than "the browser crashed when using the summarize feature on a YouTube video page."

Include Clear Steps to Reproduce

Step-by-step instructions help us recreate the issue in our testing environment. For example:

  1. Open Nav0 Browser
  2. Navigate to example.com
  3. Click on the menu icon in the toolbar
  4. Select "Summarize Page"
  5. Error message appears

Add Visuals When Possible

Screenshots or screen recordings can illustrate the issue much more clearly than text descriptions alone. Use your system's screenshot tool or a screen recording app if appropriate.

Test in Standard Mode

If possible, try to reproduce the issue with extensions disabled and in standard browsing mode. This helps determine if the issue is with the core browser or related to extensions or specific settings.

Privacy Note: All issue reports are handled confidentially by our development team. We respect your privacy and do not collect any personal browsing data through our reporting system.

Common Issues and Solutions

Before reporting an issue, check if it's a known problem with an existing solution:

Browser Performance

  • Slow performance: Try adjusting performance settings in Settings > Advanced > Performance, or disable unused extensions.
  • High memory usage: Close unused tabs and check for extensions that might be consuming excessive resources.

Browser Features

  • Features not responding: Ensure your device meets minimum requirements and try resetting in Settings > Advanced > Repair.
  • Inaccurate summaries: This can happen with complex or specialized content. Try adjusting the accuracy settings.

Website Compatibility

  • Website not loading correctly: Try temporarily disabling Enhanced Privacy Mode, as some sites rely on trackers for functionality.
  • Media playback issues: Check if the website requires DRM content that might be blocked by your privacy settings.

If the above solutions don't resolve your issue, please report it to us using the guidelines on this page.
